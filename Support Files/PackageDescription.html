EasySafari Plugin

EasySafari is a standard Joomla plugin that lets you add an icon to be displayed by the "pinned tab" feature of Safari 9.

Usage

There's only a few things to do to get up and running. The first is to create a black and white version of your icon and save it as an SVG file. Upload that file to your /images directory of your Joomla website.

  1. open the plugin and set it up
    1. select your icon's svg file
    2. specify the colour to use
    3. save your settings

That's it, nothing else is required.

Don't forget to enable and configure the plugin in the Plug-in Manager.
